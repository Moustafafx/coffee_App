# ☕ Coffee App

A Flutter app for browsing coffee drinks.

## 🚀 Features
- Browse coffee drinks
- View drink details
- Beautiful UI

## 🛠️ Tech Stack
- Flutter
- BLoC / Cubit
- Cached Network Image

## 📦 Installation
```bash
git clone https://github.com/اسمك/اسم-الريبو.git
cd اسم-الريبو
flutter pub get
flutter run
```
